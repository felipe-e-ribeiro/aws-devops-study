# Amazon SNS

**Amazon SNS (Simple Notification Service)** is a fully managed messaging service for both application-to-application (A2A) and application-to-person (A2P) communication. It uses a pub/sub model to reliably deliver messages to multiple subscribers, including AWS Lambda functions, SQS queues, HTTP/S endpoints, email, and SMS.
